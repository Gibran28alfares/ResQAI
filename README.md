# ResQAI

ResQAI adalah dashboard berbasis AI untuk menganalisis risiko bencana di Indonesia menggunakan data gempa dan cuaca secara real-time.

Aplikasi ini tidak hanya menampilkan data, tetapi mengolahnya menjadi status risiko (aman, waspada, bahaya) dan ringkasan kondisi yang mudah dipahami.

## Fitur

- Analisis risiko otomatis berbasis AI
- Data bencana 24 jam terakhir
- Data cuaca berdasarkan lokasi
- Ringkasan kondisi secara real-time

## Cara Kerja

Data bencana dan cuaca diambil dari API, kemudian dianalisis oleh AI untuk menghasilkan status risiko dan ringkasan kondisi yang ditampilkan di dashboard.

## Tech Stack

- Vite + React + TypeScript
- Express
- Gemini API
